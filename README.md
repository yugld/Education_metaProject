## Features
- Webpack 5
- Pug
- SCSS
- Babel

## Build Setup:

```bash
# Download repository:
git clone  webpack-template

# Go to the app:
cd webpack-template

# Install dependencies:
npm install

# Server with reload
npm run dev

# Server with reload at localhost:8080
npm run serve

# Output will be at dist/ folder
npm run build
```
